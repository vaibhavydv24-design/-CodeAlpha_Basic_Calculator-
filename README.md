# Basic Calculator in C

## 📌 Project Overview
This project was developed as part of the **CodeAlpha C Programming Internship (Task 1)**.

It is a simple calculator that performs basic arithmetic operations using a **switch-case** statement.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Division-by-zero handling
- User-friendly menu

## 🛠️ Technologies Used
- C Programming
- Visual Studio Code
- GCC Compiler

## ▶️ How to Run

Compile:
```bash
gcc calculator.c -o calculator
```

Run:
```bash
./calculator
```

Windows:
```bash
calculator.exe
```
